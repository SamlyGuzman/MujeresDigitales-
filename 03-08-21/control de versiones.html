Git

Sistema de control de versiones el cual nos permite trabajar en pryectos de manera organizada, el objetivo es eliminar la perdida de informacion y mejorar el trabajo en equipo.

Comandos:

git add, git commit, git checkout, git merge, git push, git pull.

github

Es una plataforma en el cual se puede trabajar colaborativamente con otras personas, gestionar proyectos, ver los diferentes versiones de un proyecto de forma facil.
